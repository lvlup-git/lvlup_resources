# lvlup_resources

a pack of resources used in our servers either created by us, or heavily edited.

# handling-gta-default

the default handling for all base gta vehicles, untouched to be edited by you, the user

# lvlup_3dme

a 3d `/me` resource which displays the message above the players head, no matter what animation their doing

# lvlup_carwash

a simple car wash script supporting r14_evidence (abandoned script)

# lvlup_core

a bunch of code jammed into 1 resource, pick it apart and use what you need, or use the whole thing

# lvlup_delivery

a delivery script that supports multiple "side jobs" for pizza delivery, and post-op delivery

# lvlup_elevator

an elevator script that allows teleporting to roofs or where ever you set

# lvlup_multijob

a re-worked multijob script from randol_multijob with enhancements, and showing the unemployed job by default

# lvlup_paycheck

a re-worked paycheck script from randol_paycheck with enhancements such as code reworks, and multi-bank support

# lvlup_preload

a teleport script that allows you to let players "preload" areas with your 1000 mlo server

# lvlup_raycast

a re-worked raycast script from v-raycast with enhancements and the ability to press "E" to copy coords

# lvlup_rental

a vehicle rental script supporting `rental_papers` with metadata (using ox_inventory)

# lvlup_trays

a tray system for business' or whatever you want to use them for

# lvlup_vehcontrol

a poorly re-worked lux_vehiclecontrol with some enhancements

# lvlup_vinchange

a script allowing vin changes for rahe-boosting (YOU NEED THE `is_stolen` CODE SNIPPET FROM HIS DISCORD)
